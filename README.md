# MS-Maos-a-Obra
M&amp;S Mãos à Obra — Construções e Remodelações em Geral (Margem Sul / Setúbal)
M&S Mãos à Obra Site institucional desenvolvido para a empresa M&S Mãos à Obra, especializada em construção, remodelação e acabamentos de alto padrão na Margem Sul. Este projeto consiste em um site estático que oferece informações sobre a empresa, seus serviços e formas de contato. O objetivo é apresentar a M&S Mãos à Obra para o público, destacando sua experiência e qualidade no setor da construção civil. A estrutura do projeto inclui o arquivo index.html, que é a página inicial, uma pasta css contendo o arquivo style.css para os estilos do site, uma pasta js com o arquivo script.js para eventuais scripts e funcionalidades, além da pasta img destinada às imagens utilizadas no site, organizada em subpastas conforme necessário. As tecnologias utilizadas são HTML5, CSS3 e JavaScript Vanilla.
